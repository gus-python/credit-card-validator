# Credit Card Validator

Script em Python que verifica a validade de números de cartão de crédito utilizando regras básicas de validação.

## Funcionalidades

O programa verifica:

- se o número possui quantidade válida de dígitos
- se o formato do cartão é válido
- validação utilizando algoritmo de verificação

## Tecnologias utilizadas

- Python

## Objetivo do projeto

Projeto desenvolvido para praticar lógica de programação, manipulação de números e validação de dados em Python.
